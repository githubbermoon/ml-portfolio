# ML Portfolio (Astro)

Cinematic ML portfolio with glassmorphism, WebGL scenes, and MDX content.

## Stack
- Astro + React islands
- Tailwind CSS
- R3F (Three.js), GSAP, Theatre.js
- Mermaid + Observable Plot

## Dev
```sh
pnpm install
pnpm dev
```

## Build
```sh
pnpm build
```

## GitHub Pages
The `base` path is set in `astro.config.mjs` for `/ml-portfolio/`.
Update it to match the repo name or remove it for custom domains.

## Cloudflare Pages
Use the static output (`dist/`). Remove `base` when you switch to Cloudflare Pages.
