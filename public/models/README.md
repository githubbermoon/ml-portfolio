Hero GLB brief
- Abstract 3D hero object for a machine learning portfolio website.
- Calm, intelligent, data-inspired form (point cloud / latent space / flowing data).

- Cinematic, minimal, serious; neutral grayscale or subtle blue tint.
- Centered at world origin (0,0,0) with uniform scale.
- No baked lighting; single or very simple material.
- Low to medium polycount.
